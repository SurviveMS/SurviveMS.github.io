# Spring Boot 总览 <!-- {docsify-ignore-all} -->

Spring Boot 是一个简化了 Spring 应用程序开发的框架。它通过自动配置和内嵌服务器，让开发者不再需要复杂的配置，简化了开发流程。

## Spring Boot 特点
- 开箱即用
- 内嵌 Web 服务器（如 Tomcat）
- 自动配置功能
